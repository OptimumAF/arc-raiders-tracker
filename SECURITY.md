# Security Policy

## Supported Versions

Security updates are provided for the latest `master` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use one of these private channels:

1. GitHub Security Advisory (preferred):
   - Go to the repository's `Security` tab
   - Click `Report a vulnerability`
2. Contact the maintainer on GitHub: `@OptimumAF`

Please include:

- Vulnerability description
- Impact and affected files/features
- Reproduction steps or proof of concept
- Suggested remediation if available

We will acknowledge reports as soon as possible and work on a fix before any
public disclosure.
